# NEURONS (NeuralNS) — обзор проекта

**Что это:** децентрализованный неймспейс для AI-агентов на Solana. Каждый агент получает читаемое имя вида `scout.agent` / `executor.defi.agent` вместо 44-значного адреса кошелька. Имя живёт он-чейн (PDA-аккаунт в нашей программе), к нему прилагается **AgentCard** — NFT-паспорт с манифестом возможностей агента (capabilities, эндпоинты, цепочки, условия оплаты).

**Зачем:** у AI-агентов в крипте нет слоя идентичности — только адреса кошельков. NEURONS делает агентов находимыми и опознаваемыми: по имени, по категории, по функции («найди всех агентов, умеющих свопы на Solana»). Аналог ENS/SNS, но построенный под машины, а не людей.

**Монетизация:** платная регистрация имён (тарифы по длине: короткие — премиум), ежегодные продления, платная верификация. Оплата в SOL или в нашем токене $NEURONS (холдеры получают скидку 25% — утилити для токена).

## Что осталось до лаунча

1. **Редеплой сайта на neuralns.xyz** (один заход на сервер) — после этого NFT-карточки начнут показывать картинку в кошельках.
2. **Запуск токена $NEURONS** → прописать его mint в конфиг + выставить боевые цены (одна транзакция).
3. Опционально (после лаунча): npm-пакет SDK, интеграции (SAID, SNS-резолвер), аудит программы, категорийные неймспейсы с роялти.

## Архитектура (кратко)

```
Пользователь (Phantom) ──подписывает tx──► Solana-программа (mainnet)
        │                                   ├─ PDA имени (owner/resolver/expiry/verified/uri)
   сайт Next.js ──► REST API (NestJS) ──►  ├─ Config PDA (цены, токен, трежери)
                       │    индексер        └─ AgentCard NFT (Token-2022)
                       └── Helius RPC (getProgramAccounts)
```

- Резолв имени работает без нашего API — любой может вывести PDA из имени и прочитать аккаунт (trustless).
- Никаких админ-ключей над чужими именами: все мутации требуют подписи владельца. Админ-ключ управляет только экономикой (цены/токен/трежери) через конфиг.
